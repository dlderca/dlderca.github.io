# [当沉默成为争吵的借口](https://dlderca.github.io/)

在亲密关系中，沉默并不总是金。它往往是“金刚钻”，因为它既坚硬又锋利。当她问你关于明天的安排时，你的回答是简短的“随便”，却不敢抬头看她的眼睛；当他发现你刷手机刷得比说话还久，他的回应不是指责，而是冷漠的回避。

这种沉默的背后，是对冲突的恐惧和对不满的压抑。双方都害怕说出心里的不快，害怕因为一句话而伤害对方。于是，沉默就成了两人之间的“屏障”，阻断了真实情感的流通。

如果你发现你们的关系正被沉默笼罩，试着主动打破僵局。不要等待情绪的爆发，而是提前设定一个“情绪检查站”。当你感觉到不安时，主动说：“我发现我们最近很少说话，是不是有什么困扰？”用柔和的语气拆解沉默的防御机制，重新建立情感的桥梁。
